Question: What is pest-opt?
Raw Content:
PESTPP-OPT is a component of the PEST++ software suite designed for decision optimization under uncertainty using sequential linear programming (SLP). It was described in detail by White et al. (2018) and builds upon methodologies initially outlined by Wagner and Gorelick (1987). PESTPP-OPT focuses on minimizing a linear objective function subject to linear constraints, where the objective function (φ=cᵀx) involves decision variables (x) and a vector of constants (c), often representing costs. The constraints are expressed as linear equations (Ax≤b), with A being a response matrix calculated either through finite differences or supplied by the user. This tool assumes the model it optimizes has been calibrated, meaning the parameters used for optimization emerge from a calibration process or are chosen based on minimized error variance from expert knowledge.

PESTPP-OPT employs sequential linear programming for efficient optimization, iterating the optimization process by recalculating the response matrix (A) to accommodate the linear approximation of model outputs to decision variables. It utilizes the open-source CLP optimization library, part of the COIN-OR project, for implementing the simplex methodology of linear programming. This approach allows handling a large number of decision variables effectively.

Additionally, PESTPP-OPT addresses uncertainties in model outputs by applying chance constraints, adjusting predictive values for uncertainty, often pessimistically. This adjustment uses a calibrated model where model outputs are near the center of their posterior distributions, considering both prior uncertainty and the reduction in uncertainty achieved through the calibration process.
