Question: How does PEST perform model calibration?
Raw Content:
1) Definition  
- PEST performs model calibration by parallelizing model runs to reduce run time, employing strategies like bounds-sticking and upgrade vector bending to manage parameter adjustments efficiently, and utilizing global optimizers like CMAES_P for handling unstable/nonlinear models without derivatives. It also offers utilities like PARREDUCE to assess the impact of parameter reduction strategies on model-data misfit.
